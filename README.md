# tools

- `autosave.py`
  - view and delete autosaves
- `jamgen.py`
  - generate a starter folder for a quake mapping jam
- `submit.py`
  - submit a quake mapping jam map as a `.zip`
- `symlink_cfg.py`
  - create symbolic links for your `.cfg` files
- `tb.py`
  - an AHK script to interface with TrenchBroom
